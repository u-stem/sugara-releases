# sugara デスクトップアプリ

旅行計画アプリ [sugara](https://sugara.vercel.app) の macOS / Windows 向けデスクトップアプリです。

## ダウンロード

**[最新バージョンをダウンロード](https://github.com/u-stem/sugara-releases/releases/latest)**

| プラットフォーム | ファイル |
|---|---|
| macOS (Apple Silicon / Intel) | `sugara_x.x.x_universal.dmg` |
| Windows | `sugara_x.x.x_x64-setup.exe` |

## インストール

### macOS

1. `.dmg` ファイルを開いてアプリを Applications フォルダにドラッグする
2. 初回起動時に「"sugara"は開いていません」と表示された場合:
   - **システム設定 → プライバシーとセキュリティ** を開く
   - 「お使いの Mac を保護するために "sugara" がブロックされました。」と表示されている
   - **「このまま開く」** をクリックする

### Windows

1. `.exe` ファイルを実行する
2. 「現在、SmartScreen にアクセスできません」と表示された場合:
   - **「実行する」** をクリックする
3. セットアップウィザードに従ってインストールする

### セキュリティソフトの警告について

コード署名を行っていないため、OS やセキュリティソフト（ウイルスバスター等）が警告を表示する場合があります。アプリに問題はありませんので、許可してインストールしてください。
